class optbeam._3d.beams.LaguerreGauss3d(x, params, called=False)[source]¶

Bases: optbeam._3d.beams.Beam3dSpherical

3d Laguerre-Gauss beam.

The implementation is based on 1.

Parameters

  • x (type) – Description of parameter x.

  • params (type) – Description of parameter params.

  • called (type) – Description of parameter called (the default is False).

References

1(1,2)

Bliokh, K Y, Aiello, A: Goos-Hänchen and Imbert-Fedorov beam shifts: an overview, Journal of Optics 15(1), 014001, 2013.

Examples

>>> LGbeam = LaguerreGauss3d(x=x, params=params)
>>> LGbeam.profile(r)
>>> LGbeam.spectrum(sin_theta, theta, phi)
